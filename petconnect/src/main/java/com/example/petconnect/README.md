# PetConnect

Sistema de adoção de cães e gatos desenvolvido para a disciplina de Implementação de Software.

## Tecnologias
- Java 21
- Spring Boot
- PostgreSQL
- JPA/Hibernate
- Maven

## Arquitetura
MVC + Service Layer

## Funcionalidades
- Cadastro de pets
- Listagem de pets
- Controle de adoção

## Como executar

1. Criar banco PostgreSQL
2. Configurar application.properties
3. Rodar aplicação Spring Boot

## GitFlow

Estratégia utilizada:
- main
- develop
- feature/*

Fluxo:
feature -> develop -> main
