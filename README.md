# 🐾 PetConnect

Sistema de adoção de cães e gatos desenvolvido para a disciplina de Implementação de Software.

## 🚀 Tecnologias
- Java 21
- Spring Boot
- PostgreSQL
- JPA/Hibernate
- Maven

## 🏗️ Arquitetura
MVC + Service Layer

## 📌 Funcionalidades
- Cadastro de pets
- Listagem de pets
- Controle de adoção

## GitFlow

Branches utilizadas:
- main
- develop
- feature/*
